# Windows-De-Mente-Black-Edition
Windows De Mente Black Edition es una herramienta de optimización, limpieza y privacidad para Windows escrita en PowerShell puro. Nació de la fusión de tres scripts desarrollados y probados durante meses — Lazarus V2, Lazarus-Nitidus y Windows De Mente v2 — tomando lo mejor de cada uno y construyendo una interfaz gráfica completa.
